# تقرير مشروع StudentTracker (Alpha)

## 1. نظرة عامة على المشروع

**الاسم:** StudentTracker (Alpha)  
**النوع:** تطبيق سطح مكتب (Windows Desktop Application)  
**الإطار التقني:** WPF (.NET 8.0)  
**اللغة:** C#  
**قاعدة البيانات:** SQLite  
**الواجهة:** العربية (من اليمين لليسار)

### الغرض من المشروع
تطبيق لإدارة وتتبع_progress الدراسي للطلاب، يشمل:
- متابعة المواد والدرجات
- حساب المعدل التراكمي
- جدولة الامتحانات
- إدارة الملاحظات
- عرض الإحصائيات والتقويم

## 4. الخدمات Services

### DatabaseService.cs
الخدمة الرئيسية للتعامل مع قاعدة البيانات SQLite. تشمل:
- **تهيئة قاعدة البيانات:** إنشاء الجداول (StudyYears, Semesters, Subjects, GradeAttempts, ExamSchedule, AppSettings, Notes)
- **العمليات الأساسية:** إضافة، حذف، تحديث، استعلام
- **إدارة الخطة الدراسية:** حفظ/تحديث الخطة
- **إدارة الدرجات:** جلب آخر درجة، إضافة درجة، سجل المادة
- **إدارة الامتحانات:** جدولة، التحقق من التعارضات
- **إدارة الملاحظات:** إضافة، حذف، تحديث

### GradeCalculator.cs
حاسبة الدرجات:
- `ComputeFinalGrade()`: حساب الدرجة النهائية (عملي + نظري)
- `ComputeStatus()`: تحديد حالة المادة
- `IsPassed()`: التحقق من النجاح (60%+)

### YearCalculator.cs
حاسبة السنة الحالية:
- `CalculateCurrentYear()`: تحديد سنة الطالب بناءً على أدائه
- **القاعدة:** النجاح التراكمي = إجمالي المواد - 4

### Logger.cs
خدمة تسجيل الأحداث والأخطاء

## 12. الإحصائيات والحسابات

### نسبة إكمال السنة:
```
(المواد الناجحة + الراسبة) / إجمالي المواد × 100%
```

### الدرجة النهائية:
```
if (إعادة_عملي) الدرجة_العملية = 0
الدرجة_النهائية = الدرجة_النظرية + الدرجة_العملية
```

### نجاح المادة:
```
الدرجة_النهائية ≥ 60%
```

### حساب السنة الحالية:
```
المطلوب للنجاح = إجمالي المواد - 4
```

## 13. التحسينات المضافة - FirstTimeSetupWindow

تم تحسين نافذة الإعداد الأولي بتجربة المستخدم (UX) والإمكانيات التالية:

### التحسينات في الواجهة:
1. **تصميم محسّن:**
   - تدرج لوني جديد (Indigo/Purple)
   - أيقونات محسّنة برؤية حديثة
   - حواف دائرية محسّنة
   - تباعد وتpadding محسّن

2. **استخدام ComboBox:**
   - قوائم منسدلة لاختيار عدد السنوات (1-6)
   - قوائم منسدلة لاختيار عدد الفصول (1-3)
   - تسميات توضيحية لكل اختيار

3. **بطاقات الفصل الدرامي:**
   - عرض كل فصل كبطاقة منفصلة
   - أيقونات دائرية showing رقم السنة/الفصل
   - ألوان متناسقة

4. **تحسين إدخال المواد:**
   - أزرار إضافة مادة بتصميم حديث
   - أزرار حذف لكل مادة
   - تأثيرات تركيز (Focus) على حقول الإدخال

5. **معلومات ونصائح:**
   - صندوق نصائح بلون أزرق
   - معلومات أسفل الشاشةshowing إجمالي الفصول

6. **أزرار محسّنة:**
   - زر إلغاء مع تأثير-hover
   - زر حفظ مع أيقونة check
   - ألوان متوافقة مع التصميم العام

### التحسينات في الكود:
1. **إدارة الحالة:**
   - حفظ عدد السنوات والفصول المختارة
   - تحديث تلقائي للبطاقات عند التغيير

2. **التحقق من البيانات:**
   - التحقق من إدخال مادة واحدة على الأقل
   - رسائل تحذيرية واضحة

3. **تجربة المستخدم:**
   - ردود فعل فورية للتغييرات
   - تأكيدdialog عند الإلغاء

## 14. الخلاصة

مشروع **StudentTracker** هو تطبيق WPF متكامل لإدارة المسار الدراسي للطلاب، يتميز بـ:
- واجهة عربية سلسة
- قاعدة بيانات SQLite محمولة
- حساب تلقائي للسنة والدرجات
- دعم كامل للمخطط الدراسي متعدد السنوات
- إحصائيات وتقويم تفاعلي
- تصميم حديث ومحسّن لتجربة المستخدم
